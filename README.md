# Match Hub 🎲💑

Una aplicación móvil de juegos interactivos para parejas, fiestas y aventuras compartidas. Construida con React Native y Expo.

## 📱 Descripción

Match Hub es una app que reúne diferentes módulos de juegos para fomentar la conexión y diversión en grupo. Actualmente incluye el módulo de **Pareja** con cartas íntimas para conocerse mejor, y próximamente módulos para **Fiesta** y **Aventura**.

### 🎯 Características Principales

- **Módulo Pareja**: Cartas románticas con preguntas profundas y retos divertidos
- **Selección Personalizada**: Elige cuántas preguntas quieres responder (5-50)
- **Interfaz Moderna**: Diseño responsivo con animaciones suaves y gradientes
- **Puntuación Dinámica**: "Conectamos" vs "No conectamos" para medir compatibilidad
- **Módulos Expansibles**: Arquitectura preparada para nuevos juegos

## 🚀 Instalación

### Prerrequisitos

- Node.js (versión 16 o superior)
- npm o yarn
- Expo CLI (`npm install -g @expo/cli`)
- Un dispositivo móvil o emulador

### Pasos de Instalación

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/juankAnez/AppMatch.git
   cd AppMatch
   ```

2. **Instala las dependencias:**
   ```bash
   npm install
   ```

3. **Inicia la aplicación:**
   ```bash
   npx expo start
   ```

4. **Escanea el código QR:**
   - Con la app Expo Go en tu teléfono
   - O usa un emulador

## 🎮 Cómo Jugar

1. **Selecciona un Módulo**: Elige entre Pareja, Fiesta o Aventura
2. **Configura el Juego**: Ingresa los nombres de los jugadores y el número de preguntas
3. **Juega**: Responde las preguntas y marca si conectan o no
4. **Ve los Resultados**: Revisa el porcentaje de conexión al final

## 🛠️ Tecnologías Utilizadas

- **React Native**: Framework para desarrollo móvil
- **Expo**: Plataforma para desarrollo y distribución
- **Expo Linear Gradient**: Para efectos de gradiente
- **Expo Vector Icons**: Iconos de Ionicons
- **JavaScript ES6+**: Lenguaje de programación
